# 车队维修保养与备件领用服务

任务类型：0-1代码生成

业务领域：纯后端服务

修改范围：跨模块多文件

任务难度：中等偏难

题面：

实现一个车队维修保养与备件领用服务，车辆按里程、日期和故障等级生成保养计划，维修人员创建工单后可登记诊断、工时、备件消耗和完工结果。正常工单要能从待派工流转到维修中、待验收和已完成，并自动更新车辆健康档案。车辆仍在运行中开维修单、备件库存不足、里程读数倒退、未验收工单直接完成、重复领用同一备件都要拒绝。服务提供车辆健康档案、保养计划查询、备件库存预警和维修明细导出，导出包含工时、备件、故障原因和验收结果。重启后工单、里程、备件流水和导出记录保持一致。验收还要覆盖保养计划自动生成、工单作废后备件退回。
